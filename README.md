# Color Name Finder

Enter any hex color code and find the closest matching color name.

## How to Use

**Step 1:** Open `index.html` in your browser.

**Step 2:** An onboarding tutorial appears on first visit. Click **Open Finder** (or Skip).

**Step 3:** Enter a hex code (like `#ff6b6b`) in the text field, or click the color picker.

**Step 4:** Click **Find** (or it auto-detects as you type).

**Step 5:** The closest color name appears with a preview swatch.

**Step 6:** View HEX, RGB, and HSL values. Click any value card to copy it.

**Step 7:** Click **Random Color** to discover new colors.

**Step 8:** Recent colors are saved in the history below.

## Features
- 80+ named colors database
- Closest color name matching (Euclidean distance)
- Color picker and hex input
- HEX, RGB, HSL value display
- Click to copy any value
- Random color generator
- Recent colors history
